Method & data sources

How these numbers are produced, segmented, and validated.

One pipeline → one database

Every number on this site comes from a single deliverable, kotook_market.duckdb, produced by the kotook_pipeline notebook from the latest DLD open-data dump. To refresh each year: drop the new DLD CSVs into puls_data/ and run the notebook — it auto-picks the newest valid file per dataset, recomputes every statistic, runs a validation gate, and writes the database + CSV/parquet exports. This site then just renders that database.

Off-plan vs secondary vs combined

Every statistic is split three ways using the DLD reg_type_en field: off-plan (primary / developer sales), secondary (ready / resale), and combined. Off-plan is 59% of sales since 2023 — Kotook's core market. Price growth (YoY) is read off the secondary segment, which strips off-plan launch-mix noise.

Source & the validation field

Dubai Land Department public Transactions — 1,274,696 Sales records (mortgages/gifts excluded). Area mapping is validated against the dominant master_project_en inside each DLD area; developer attribution follows the official project_number → developer chain.

Coverage

Area mapping covers 90% of all sales (82 Kotook communities); the rest are listed on the Areas page as review candidates. Developer attribution covers 49.0% of sales (the share linked to a DLD-registered project).

Data cleaning & outliers

  • No duplicates / no missing IDs — every transaction has a unique transaction_id.
  • Price metrics use a clean residential basis — median AED/sqft, YoY, off-plan premium and share use only Unit/Villa in a plausible AED/m² band (drops Land + commercial + impossible prices, ~9.7% of rows). Because we use the median, headline numbers barely move — the cleaning is for correctness, not to chase a number.
  • Value sums stay over all sales, but bulk/portfolio deals above AED 500M (~4.5% of the 2023+ total) get an "excl. bulk >500M" figure on each value card.
  • Counts always cover all registered sales.

Rental yield (Ejari)

From the DLD Rent Contracts (Ejari) registry (1,146,844 residential contracts in 2025). Gross yield = median residential rent per sqft ÷ median secondary/ready sale price per sqft for the same year. Ejari only records rents of existing stock, so it is paired with the ready price (the apples-to-apples read); the all-market price (which carries the off-plan premium) understates yield and is shown only for context. It is a gross figure — service charges and vacancy are not deducted. City gross yield: 5.1%.

What to validate

For each area and developer we list 20 recent transactions with their segment. Confirm that project / building / master names genuinely belong to the stated Kotook community or developer. In Kotook, the area mapping is stored on CityArea.old_name (en).
